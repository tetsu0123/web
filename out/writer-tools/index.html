Writer's Tools

執筆活動を支援する各種変換・整形ツール

総合変換ツール

複数の変換機能を統合した万能ツール。改行削除、字下げ、文字数カウント、全角変換を一箇所で実行。

Main Tool

文字数カウンタ

原稿の文字数をリアルタイムでカウント。空白・改行を除いた純粋な文字数と原稿用紙換算を表示。

Counting

字下げツール

段落の先頭に全角スペースを自動挿入。日本語文章の正しい体裁を整えます。

Formatting

改行削除ツール

不要な改行を削除して文章を整形。段落の区切りは保持しながら無駄な改行を除去。

Cleaning

半角→全角変換

半角英数字を全角に変換。日本語文章における統一感のある表記を実現。

Conversion

テキスト→HTML変換

プレーンテキストをHTMLに変換。改行をbrタグに、段落をpタグに自動変換。

HTML

テキスト統合

複数のテキストファイルを一つに統合。章ごとに分かれた原稿をまとめる際に便利。

File Operation

PDF結合

複数のPDFファイルを一つのファイルに結合。原稿の提出や配布に便利。

PDF

データ保存

作業中のテキストを一時保存。ブラウザのローカルストレージを利用した簡易保存機能。

Storage

シナリオフォーマット

脚本・シナリオ執筆のための専用フォーマットツール。ト書きや台詞の整形に対応。

Screenplay
